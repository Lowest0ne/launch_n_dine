= Launch 'n Dine
== An automous food delivery service
== How to use
* fork and clone
* bundle
* create a .env in rails root with your secret token
* create and migrate database
== Rails and Ruby Versions
* Rails 4.0.0
* Ruby 2.0.0p247
== Database
* psql (PostgreSQL) 9.2.4
== Test Suit
* rspec 2.13.1
